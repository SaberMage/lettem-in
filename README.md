## So yer tired o' pickin' up yer phone, are ye?
*Have your Android phone answer any caller you want, any way you want. All you need is a Teensy 4.1 development board plugged into your phone's USB port!*

### Features
- Customizable profiles
  - Multiple contacts per profile
  - Catchalls for `anonymous`, `likely spam`, and `not-in-contacts` callers
- Different behavior modes:
  - `Reject`
  - `DTMF` (dialtone)
  - `Audio`
  - `Audio-then-DTMF`
- Choose your dialtone sound
- Choose any MP3 or WAV file from the app
- Set audio file volume (most files work best between 10-15%)

### Use Cases
- Mine: auto-answer the callbox to unlock the door for a guest or delivery driver
- Funny sound bites when friends or family call
- Also mine: reject spam callers *(doesn't need the Teensy + never rings!)*

For easy installation and keeping up with the latest version, I highly recommend [Obtainium](https://github.com/ImranR98/Obtainium).
